预备知识

集合

  • 互异性、无序性

  • 列举法、描述法

  • 包含与相等：

    • 包含：$A \subseteq B \iff \forall x \in A, x \in B$
    • 不包含：$A \nsubseteq B \iff \exists x \in A, x \notin B$
    • 相等：$A = B \iff A \subseteq B \land B \subseteq A$
    • 不相等：$A \neq B \iff A \nsubseteq B \lor B \nsubseteq A$
    • 真包含：$A \subset B \iff A \subseteq B \land A \neq B$
  • 集合运算：
    • 并集：$A \cup B = {x | x \in A \lor x \in B}$
    • 交集：$A \cap B = {x | x \in A \land x \in B}$
    • 相对补：$A - B = {x | x \in A \land x \notin B}$
    • 绝对补：$A^c = E - A = {x | x \notin A}$
    • 对称差：$A \oplus B = (A - B) \lor (B - A)$
